# Orchard Ten 平衡報告

產生日期：2026-07-16
固定 seed：`20260716`

## 方法

- Legacy 與 Balanced Classic 各分析 1,000 個開局棋盤。
- 使用相同 seed、相同全矩形前綴和分析器與相同 10%／25%／50%／75% 進度模擬。
- 每個版本另執行新手、一般、熟練玩家模型各 300 局。
- Quick、Zen、Hard 各分析 300 個棋盤及每種玩家模型 100 局。
- 模擬玩家是可重現的版本比較工具，不代表真人行為。

## 最終規則

- 棋盤：最多 24 次受約束候選生成，再使用隨機多區保底；每次回傳品質 metadata。
- 品質：可解性 20%、多樣性 15%、空間分布 15%、重疊 12%、開局友善 14%、中後期延續 16%、得分潛力 8%。
- 提示：只高亮一組中等複雜度有效矩形，不直接消除；無解時不扣次數，位置由候選池隨機選擇。
- 重排：確認無解且至少剩兩顆水果時，系統立即免費自動重排；保底會把剩餘水果重組為隨機的連續湊十群組，避免反覆只保證下一步。分數與時間保持不變，玩家無法主動重排。
- 選取：空格視為 0，有效矩形可以跨越已消除格；只清除並計算矩形內仍存在的水果。
- 計分：每消除一顆水果固定 1 分；沒有矩形大小加分、Combo 倍率或其他分數來源。
- 隱私：工具只處理本機合成棋盤，不記錄滑鼠軌跡、不傳送資料、不加入第三方分析。

## 中後期風險

空格視為 0 並允許跨越後，中期延續性明顯改善：50% checkpoint 的無解率由 Legacy 的 2.4% 降至 Balanced Classic 的 0.1%，一般模型平均系統重排只有 0.04 次。到 75% checkpoint 時水果數量已很少，無解率仍為 94.0%，略高於 Legacy 的 90.2%；實際規則以免費系統重排處理，熟練模型平均重排由 Legacy 的 13.51 次降至 1.00 次。後續真人測試應確認極後期的單次重排是否自然，以及高階玩家在純消除計分下是否仍有足夠成就感。
